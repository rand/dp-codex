# Agent Session Bootstrap

First command:

```bash
dp agent bootstrap --json --detail brief
```

Then:

1. Follow any instruction audit next action before editing.
2. If a campaign is present, run the suggested campaign status or loop command.
3. If adoption is incomplete, plan before applying.
4. Use `dp agent capabilities --json` when command safety or mutability is unclear.
5. Expand with `--detail full` only when the brief response omits needed detail.
